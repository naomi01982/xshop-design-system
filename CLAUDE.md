# Xshop Design System Instructions

このリポジトリは、Xshop株式会社の公式デザインシステムです。

Claude DesignでXshopのWebサイト、LP、営業資料、スライド、提案書、SNS画像、採用資料を作成・修正するときは、必ずこのデザインシステムに従ってください。

## Brand Tone

Xshopは、飲食・サービス業の現場を深く理解した店舗DX企業です。

表現は以下を重視します。

- 現場目線
- 結論先行
- 実直
- 信頼感
- 利益改善に直結する表現
- 飲食店、小売店、サービス業の経営者・本部担当者・現場スタッフに伝わる具体性

避ける表現：

- 抽象的なDX表現
- かっこよさ優先の曖昧なコピー
- 機能羅列だけの説明
- 過度にスタートアップっぽい軽い表現
- 「革新的」「次世代型」など中身が伝わりにくい表現

## Color Tokens

- Primary Navy: #003D99
- Deep Navy: #002B6B
- Secondary Blue: #1A6FD4
- Accent Red: #E22B13
- Highlight Yellow: #FFE000
- LINE Green: #06C755
- Light Blue Background: #E8F4FB
- Paper Background: #F4F6FA
- Card White: #FFFFFF
- Border Gray: #E2E7EF
- Main Text: #1C2430
- Soft Text: #5A6472

## Color Usage

基本は「白・ライトブルー・ネイビー」で構成します。

- レッドは最重要強調のみに使う
- イエローは章見出しや限定訴求のみに使う
- LINE GreenはLINE連携の説明だけに使う
- 装飾目的で色を増やさない

推奨比率：

- 60%：白・ライトブルー
- 30%：ネイビー
- 10%：アクセントカラー

## Typography

- Japanese Font: Noto Sans JP
- English Font: Inter

見出しは太く、本文は読みやすく短く区切ります。

## Layout

8pxグリッドを基本にします。

余白の基準：

- 4px
- 8px
- 16px
- 24px
- 32px
- 48px
- 64px

基本構成：

1. 課題
2. 解決策
3. 機能
4. 効果
5. 実績
6. CTA

## Components

- Primary CTA: ネイビー背景 + 白文字
- Important CTA: レッド背景 + 白文字
- LINE CTA: LINE Green背景 + 白文字
- Card: 白背景、薄いグレー枠、角丸、軽いシャドウ
- Table: ネイビーヘッダー + 白 / ライトブルーのゼブラ行
- Badge: 特許出願中、多店舗対応、LINE連携、導入実績などに使用

## Illustration

HP用イラスト：

- 顔のある親しみやすい人物
- ネイビーまたはインク色の線
- 淡い塗り
- フラットで軽やか
- 強い影や立体感は使わない

資料用アイコン：

- シンプルな線アイコン
- 機能や手順が一目で分かる
- 必要な1色だけ淡く差す

写真：

- 実績、信頼、現場感を伝える場合に使用
- 飲食現場は暖色、湯気、ライブ感を重視
- 粗い画像は使わない

## Copy Rule

良い表現：

- シフト作成にかかる丸一日の作業を、数時間に短縮します。
- 現場の提出しやすさと、本部の管理しやすさを両立します。
- LINEで提出、管理画面で一元管理。店舗運営のムダを減らします。

避ける表現：

- 革新的なDXソリューションを提供します。
- 次世代型の店舗運営を実現します。
- 多機能で便利なシステムです。

## Output Rule

Xshop関連の制作物を作るときは、必ずこのデザインシステムを前提にしてください。

提案時は、色・余白・見出し・CTA・トーンがXshop Design Systemに合っているか確認してください。

## Logo Usage Rules

- ロゴの縦横比は変更しない
- ロゴの色は変更しない
- ロゴに影、光彩、グラデーションなどの装飾を追加しない
- ロゴ周囲には十分な余白を確保する
- 背景が暗い場合は、視認性を確認して使用する
- サービスごとの資料・LP・提案書では、該当サービスのロゴを優先して使用する
- 複数サービスを並べる場合は、ロゴサイズを揃える
- Xshop全体を表す場合は、`xshop-symbol-mark.png` または `xshop-symbol-mark-wide.png` を使用する
